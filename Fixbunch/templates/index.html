About the tool

The Fixbunch algorithm uses custom-made facial recognition software to identify and organize pictures of a person in a database. For example; this tool can be used to find your prom selfies in the endless sea of pictures.

Facial detection

Facial recognition software is a technology capable of identifying a person from a digital image. The software works by comparing selected facial features from given images with faces within a database. The facial recognition software is preprogrammed to detect faces when partially visible, either due to the position or when specified person is wearing facial coverage.

High accuracy

The program was tested by using the algorithm on 4 well-known databases, and averaging their precision. Total model parameters exceeding 23 million. How does it work?

Upload a folder with the pictures

Simply start by uploading a folder with the pictures database. The uploading of the image database does not hold any constraints.

Upload a picture of the person you want to identify

Upload a picture of the person you want to find in the folder. Make sure the face is clearly visible and face towards the camera.

Wait

Have a little patience and wait for the algorithm to finish. The bigger the uploaded dataset, the longer it takes the algorithm to complete.

Download the sorted pictures

The output of the algorithm will be presented in the form of a downloadable folder. Simply download this folder and the desired pictures are on your computer!
